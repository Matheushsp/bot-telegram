# bot-telegram 
# 📌 Bot Validador de CPF no Telegram

## 📖 Sobre o Projeto

Este projeto foi desenvolvido como atividade acadêmica com o objetivo de aplicar conceitos fundamentais de programação por meio da criação de um bot no Telegram capaz de realizar a validação de números de CPF.

O desenvolvimento foi realizado utilizando comunicação direta com a API oficial do Telegram, sem a utilização de bibliotecas que encapsulem ou abstraiam suas funcionalidades.

# 🤖 Desenvolvimento Prático

## 📌 Objetivo do Bot

O bot desenvolvido possui a função de validar números de CPF enviados pelos usuários através do Telegram.

O sistema realiza:

* Recebimento da mensagem enviada pelo usuário
* Tratamento das informações recebidas
* Validação do CPF informado
* Retorno automático informando se o CPF é válido ou inválido

# 🚀 Funcionamento do Bot

## 1️⃣ Criação do Bot

O bot foi criado utilizando o BotFather.

---

## 2️⃣ Comunicação com a API do Telegram

A comunicação ocorre diretamente através da API oficial do Telegram utilizando requisições HTTP para:

* Receber mensagens
* Processar informações
* Enviar respostas automáticas

---

## 3️⃣ Processo de Validação

Quando o usuário envia um CPF:

1. O sistema recebe a mensagem
2. Os dados são tratados
3. O CPF é analisado
4. Os dígitos verificadores são calculados
5. O resultado é retornado ao usuário
